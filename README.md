# NoActive2-Mod
在NoActive v3正式發佈前體驗更好的NoActive

## 目前與NoActiveV2原版的區別
1. 流暢度增加
2. 修復MiPush通知無法喚醒進程
3. FCM推送臨時解凍
4. 網絡解凍(僅限MIUI)
5. 進程音視頻播放解凍
6. Android 14 兼容性更新
7. 支持多開應用凍結
8. 支持關閉日誌

## 提示
與最新版NoActiveV2和內測版NoActiveV3配置文件不兼容
