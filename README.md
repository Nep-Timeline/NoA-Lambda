# NoActive-λ
基于NoActive v2的修改版本，获得更好的体验

## 目前與NoActiveV2原版的區別
1. 流暢度提高
2. 通知喚醒
3. FCM推送臨時解凍
4. 網絡解凍(僅限MIUI)
5. 常驻通知检测
6. 冻结Binder
7. 压缩进程
8. 绝对冻结
9. 通知交互
10. 進程音視頻播放解凍
11. Android 14 兼容性更新
12. HyperOS兼容性更新
13. 支持多開應用凍結
